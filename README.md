# Zomato Restaurant Data Analysis & Rating Prediction

## Problem Statement
The objective of this project is to analyze restaurant metadata and customer reviews to understand customer behavior, extract insights, and build machine learning models to predict restaurant ratings.

## Dataset
- Restaurant metadata (cost, cuisines, collections)
- Customer reviews and ratings

## Methodology
- Data Cleaning & Wrangling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Text Preprocessing & TF-IDF Vectorization
- Hypothesis Testing
- Machine Learning Models (Linear Regression, Random Forest, SVR)
- Model Evaluation & Hyperparameter Tuning

## Models Used
- Linear Regression
- Random Forest Regressor
- Support Vector Regression (SVR)

## Evaluation Metrics
- RMSE
- RÂ² Score

## Results
Random Forest Regressor achieved the best performance with improved prediction accuracy and generalization.
